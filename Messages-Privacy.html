Privacy Policy

Message Privacy

As messages are delivered to client, they pass through the central server, these messages are end to end encrypted so only users can access the contents.

The server does not have the ability to send messages by itself or access the contents of user's messages

Collected Data

Messages are stored in firebase until delivery in which they are removed from the system entirely.

User identifying tokens are stored on the server, along with data allowing for the server to connect to APNS on the user's behalf

Users interact

All messages come from other users, by using the app you recognize that messages are not filtered or monitored by the app
